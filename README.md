vim-systemd-syntax
==================

Syntax highlighting for systemd service files in Vim.


Installing
----------

* Using [Vundle](https://github.com/gmarik/vundle), add the following to your `.vimrc` and
  `:BundleInstall`

        Bundle "Matt-Deacalion/vim-systemd-syntax"
